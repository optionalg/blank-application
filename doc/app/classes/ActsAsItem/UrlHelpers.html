ajax_items_path(model)

Dynamically create Item path for given Item for Pagination

Usage:

ajax_items_path(Article)

will return "www.example.com/workspaces/1/ajax_content/Article" inside workspace

else will return "www.example.com/ajax_content/Article"

current_workspace()

Check for Worksapce

can be called from anywhere to check if the user is inside a workspace.

Will return the workspace object if workspace parameter exists else will return nil
