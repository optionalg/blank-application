BLANK APPLICATION

Multimedia Content Management System.

Author: ThinkDRY

Version: 1.02

Requirements

Ruby : 1.8.6 or 1.8.7

RubyGems: 1.3.0 or higher

Rails: < 2.3

DB: Mysql, SQLite

Other Packages:

Xapian Package for Full text Xapian Search

sudo apt-get install libxapian15 libxapian-ruby1.8

Rfeedparser for Feed Sources

sudo apt-get install libxml-ruby1.8 libxml-parser-ruby1.8

Get a Copy
 - Git Repository thinkdry@dev.thinkdry.com:/home/git/blank.git (need SSH access to Server)
 - Send a mail to the thinkdry team at team@thinkdry.com
 - Download from Thinkdry WebSite
Installation & Setup

1) Unzip the blank.zip package in the desired folder

2) Create a database.yml file in /config/ 

   Settings can be put depending on your Database for EX: for Mysql

   production:
    adapter: mysql
    encoding: utf8
    database: blank_production
    username: mysql_user_name
    password: mysql_password
    socket: /var/run/mysqld/mysqld.sock

   to get the socket path go to mysql prompt and type "status"
   copy the UNIX socket path.

3)
